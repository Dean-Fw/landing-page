About me

Hello, world! My name is Dean, a software engineer from the UK. I've worked primarily on backend projects and creating Infrastructure as Code solutions to host them. This has involved becoming very familiar with .NET with C#, T-SQL, Terraform, and Azure DevOps pipelines.

While I spend most of my professional time working on the backend, I was professionally trained to be a full-stack engineer via a graduate scheme. This involved working on a few full-stack projects using React, .NET with C# and T-SQL. I have felt the front-end side of that training rust a bit. So I thought, why not put together a static site to brush those skills off while also improving my knowledge of building and deploying IaC solutions by hosting it!

This Site!

The IaC for this site is written in Terraform, which configures a standard storage account and a static site container to host the website files. The site is composed solely of HTML and CSS, with a touch of JavaScript. When building the site I wanted to focus on responsive CSS design, leading me to learn a lot about container and media queries, along with custom CSS properties.

To deploy my IaC and Static site files, I chose to use GitHub Actions. The technology was new to me, as I in my working life I have only really used Azure DevOps pipelines to set up automated jobs. Actions are ran from YAML workflow files which felt to be a more event based way of triggering your jobs. Overall I quite like a lot of the aspects of Actions and Workflows and hope to improve on my skills in the area through creating future projects.
